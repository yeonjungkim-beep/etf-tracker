⚙️ Google Apps Script 연결 설정
1. Google Sheets 새 파일 생성
2. 확장 프로그램 > Apps Script 열기
3. Code.gs 코드 전체 붙여넣기 후 저장
4. 배포 > 새 배포 → 유형: 웹앱
  실행: 나(본인) / 액세스: 모든 사용자(익명 포함)
5. 배포된 웹앱 URL을 아래에 붙여넣기
6. Apps Script 편집기에서 setupTrigger() 1회 실행
